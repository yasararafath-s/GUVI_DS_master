# DataSpark: Illuminating Insights for Global Electronics

## Project Overview
DataSpark is a data analytics project focused on deriving actionable business insights from Global Electronics' sales, customer, product, and store data.

## Objectives
- Analyze customer demographics and purchasing behavior
- Evaluate product performance and profitability
- Assess store-level sales performance and operational efficiency
- Understand the impact of currency exchange rates on international sales

## Tools & Technologies
- **Python** (Pandas, Matplotlib, Seaborn) – Data cleaning & EDA
- **MySQL** – Data storage and SQL-based analysis
- **Power BI** – Interactive dashboards and visualizations

## Files
| File | Description |
|------|-------------|
| `01_initial.ipynb` | Data cleaning, preprocessing & EDA notebook |
| `sql_code.sql` | SQL queries for data analysis |
| `dataspark_GE.pbix` | Power BI dashboard file |
| `dataspark_GE.pdf` | Exported PDF of the dashboard |
| `csv_data/DataSet/` | Raw CSV datasets (Customers, Products, Sales, Stores, Exchange Rates) |

## Datasets
- **Customers.csv** – Customer demographics
- **Products.csv** – Product catalog and pricing
- **Sales.csv** – Transactional sales data
- **Stores.csv** – Store location and metadata
- **Exchange_Rates.csv** – Currency exchange rates
- **Data_Dictionary.csv** – Column descriptions for all datasets

## Key Insights
- Customer segmentation by age, gender, and geography
- Top-performing products and categories
- Store-wise revenue and profitability trends
- Currency impact on global revenue
