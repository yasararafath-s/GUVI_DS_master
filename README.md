# GUVI DS Projects – Combined Repository

This repository consolidates all Data Science projects completed as part of the **GUVI ITMDSA MDE96 [DS-WD-E-B9]** program into a single organized workspace.

---

## 📁 Projects

| # | Project | Description |
|---|---------|-------------|
| 01 | [DataSpark – Global Electronics Insights](./01_DataSpark/) | EDA and dashboards for a global electronics retailer using Python, MySQL & Power BI |
| 02 | [ITMDSA MDE96 Final Project](./02_ITMDSA_MDE96_Final_Project/) | Capstone project integrating data science and web development |
| 03 | [Car Dheko – Used Car Price Prediction](./03_DS_Car_Dheko/) | ML regression model to predict used car prices with a Streamlit app |
| 04 | [Dominos – Predictive Purchase Order System](./04_Dominos_Predictive_Purchase/) | Time-series forecasting for ingredient demand and purchase order generation |
| 05 | [Microsoft – Classifying Cybersecurity Incidents](./05_Microsoft_Cybersecurity/) | Multi-class classification of SOC incident triage grades using the GUIDE dataset |
| 06 | [RedBus – Bus Route & Booking Analysis](./06_RedBus/) | Web scraping, MySQL storage & Streamlit app for RedBus travel data |

---

## 🛠 Common Tech Stack

- **Languages**: Python, SQL
- **Libraries**: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, XGBoost
- **Databases**: MySQL
- **Visualization**: Power BI, Streamlit
- **Web Scraping**: Selenium

---

## 🚀 Getting Started

Each project lives in its own subfolder with a dedicated `README.md`. Navigate to the relevant folder and follow the setup instructions there.

```bash
# Example: set up a project
cd 03_DS_Car_Dheko
pip install -r requirements.txt
streamlit run app.py
```
